1. Resumen

On this challenge, we learned to discover password with three tools: pydictor, dymerge and hydra. Also we tried to attack an user on localhost, and the we tried to a attack a web page.

2. Conceptos Aprendidos
  • Hydra: Hydra are a tool that execute an attack to some user or login of a web page to crack the password.
  • Dymerge: Dymerge is a tool whose function is merge two dictionaries of pydictor, for example merge usernames and passwords in a unique document.
